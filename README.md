# Adonis API + User Authentication

## Basic Features:

- User login
- User Authentication
- Once the user is authenticated, can perform the below operations:
  - Add an item
  - Delete an item
  - Update an item
  - Create an item

## Technologies used:

- Adonis.js
- MySQL
- Scrypt Hashing

## Notice:

- Make sure to create the database and configure it in the .env file
